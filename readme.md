# ExpansesMVVM

This app demonstrate how to use MVVM archtecture in android with following libraries.

Libraries : 
* Hilt-Dagger
* Coroutines & Flow
* ViewModel
* LiveData
* Room
* Retrofit
* Navigation
* Data-Binding
